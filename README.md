# Central-DB-Backup

**Usage of Script**

    1. Script takes the backup of existing DB's.
    2. It stores the dump files in specific directories.
    3. Deletes the 30 days older files from the specific directories.
    
**How to run**

     ./Central-DB-Backup.sh
